# IZV-ISST_online_tour
The program compiles panoramic (360-degree) images taken with a mobile phone, depicting the premises of RTU IZV and ISST (Āzenes Street 12, Riga, Kurzeme District, 4th floor). The website displays these images and allows the user to navigate by clicking arrows that are either embedded in the image itself or exist only in the user interface.

# Palaišanas pamācība:
1. Atver termināli jebkurā izstrādes vidē.
2. Programmu palaiž ar komandu "python -m http.server" (pārliecinies, ka datorā jau ir python vide)
3. Terminālī nolasa "porta" numuru (visdrīzāk 8000).\
4. Pārlūkprogrammā atver http://localhost:8000/lapa.html.


Marzipano rīka dokumentācija : https://www.marzipano.net/docs.html un https://www.marzipano.net/reference/.
